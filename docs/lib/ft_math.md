### ft_pow
Calculate the value of `x` raised to the power of `n`.

```c
/*
 * Calculate x raised to the power of n
 * 
 * @param1  double number to raise
 * @param2  double exponent
 * @return  double calculated number
 */
double  ft_pow(double x, double n);
```
